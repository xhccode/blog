z-index基础

			含义：
				z-index 属性指定了元素及其子元素的 [ z顺序 ]，而 [ z顺序 ] 可以决定当元素发生覆盖的时候，那个元素在上面。
			通常一个较大 z-index 值的元素会覆盖较低的哪一个。
			
			属性值：
				z-index：auto；  			默认值
				z-index：；  	整数值  1,2,3
				z-index：inherit；  		继承
			
			特性：
				1.支持负值；-1
				2.支持 css3 animation 动画
				3.支持 css 2.1 时代，需要和定位元素配合使用；
					备注：
						如果不考虑 css3 ，只有定位元素 （ position:relactive / absolute / fixed / sticy） 的 z-index 才有作用个！
					在 css3 中有例外 ...
			
			
z-index 与 css 定位属性

			
			position：static;z-index:2;			×
			position：relactive;z-index:2;		√
			position：absolute;z-index:2;		√
			position：fixed;z-index:2;			√
			position：sticky;z-index:2;			√
